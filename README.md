# csv-maker
This is a CSV Maker! where you can create or edit CSVs in a Python GUI.

#Description
This is a CSV maker(kinda like Excel but as GUI) where you can create .csv's, as well as editing exisiting ones. You can also add or remove rows and columns, clear all data(rows only, not columns) use a searchbar, and save CSV's

#Features:
- Create and edit CSV files
- Add/delete rows and columns
- Clear all data rows
- Search bar to filter data
- Save CSV files

#Installation
1. Open up cmd
2. Paste https://github.com/Iamthatguytoo/csv-maker.git 
3. Change directory with cd csv-maker
4. Create a Python Environment with python -m venv venv
5. Activate it with venv\Scripts\activate(or .\venv\Scripts\activate.bat if it doesnt work)
6. Install the required libraries with pip install -r requirements.txt
7. Now run it with python main.py
